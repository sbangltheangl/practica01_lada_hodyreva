
# Repair Service

Учебный проект информационной системы для сервиса ремонта техники.

## Возможности
- регистрация;
- авторизация;
- создание заявок;
- просмотр заявок;
- админ-панель.

## Технологии
- HTML;
- CSS;
- JavaScript;
- PHP;
- MySQL.

## Запуск проекта
1. Установить XAMPP/OpenServer.
2. Импортировать database.sql в phpMyAdmin.
3. Поместить проект в папку htdocs.
4. Запустить Apache и MySQL.
